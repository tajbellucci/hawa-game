# Hawa's Game

Offline single-file HTML toddler learning game. Teaches shapes and colors through interactive play — built for Hawa (age 2-4).

## Status

**Complete** — Fully functional single-file game. Works offline in any browser. Zero dependencies.

## Features

- **Shape Recognition** — Touch/click shapes to learn names (English + Urdu)
- **Color Learning** — Identify and match colors
- **Sound Feedback** — Cheerful audio cues for correct answers
- **No Pressure** — Unlimited attempts, no scoring/failure states
- **Bilingual** — English and Urdu labels and voice guidance
- **Offline** — Works without internet

## How to Play

1. Open `index.html` in any browser
2. Touch a shape to hear its name
3. Match shapes or colors as prompted
4. Celebrate correct answers with sound/animation

## Technical Details

- **Language**: HTML5 + Canvas + Web Audio API
- **Framework**: None (vanilla JS)
- **Size**: Single ~40KB file (assets embedded)
- **Performance**: 60fps animation
- **Accessibility**: Large touch targets, high contrast

## Design Philosophy

- **Distraction-free** — No ads, tracking, or UI chrome
- **Low-frustration** — Can't fail, only learn
- **Durability** — Works the same in 2026 as 2024
- **Portable** — Email the file, play anywhere

## Customization

Edit `index.html` to:
- Change shapes/colors
- Add new languages
- Adjust difficulty (shape complexity)
- Modify sound effects
- Change background colors
